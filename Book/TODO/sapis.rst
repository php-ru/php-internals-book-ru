Server Application Programming Interfaces : SAPI
================================================

When you want to ask PHP something, you need to contact it. This sound silly,
but there are different ways of contacting PHP, different entry points. These
are called SAPIs. This chapter will help you understand how they work, what
their responsibilities are and finally, how to create your own, should you
need it.
